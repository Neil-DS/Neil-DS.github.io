Cooking Innana

Cooking Inanna was a challenge to code each day for 100 days, I choose to focus on graphics programming by attempting a 3D game and seeing what I could produce within 100 days.

Starting the project I had no expectations, only some guides I knew I was going to rely on for learning OpenGL and graphics programming and the idea of making a 3D game, I wanted to go in with the mindset of learning and the desire to learn guiding me to interesting problems to solve. with this mindset I went on to re-learnt a lot of the foundations of programming with this project, importance of writing code in a clear, understandable and concise manner with plenty of comments. Along with graphics programming, GLSL, code architecture, and lots more general programming skills.
