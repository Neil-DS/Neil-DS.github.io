Greetings Mortal

Welcome I'm Neil De Swart a junior games programmer living in Scotland, Dundee. This website is still a work in progress so if you've found your way here please bare with me.

HMS Unicorn

A project undertaken on behalf the HMS Unicorn museum in Dundee, Scotland. The goal was to create an Educational and Fun exhibit, teaching visitors about the kinds of supplies required to sail aboard an 19th Centuary Warship.

Woven Together

A joint project between Dundee University and the Abertay Historical Society. In an effort to show multiculturalism in Dundee, we were tasked with creating a cooking game using a provided recipe for Nigerian Fried Rice.

Bad Old Man

2D game made in the Unity engine, players goal is to delve, loot and redelve for more loot crafting and enchanting items along the way. and possibley getting a few adventurers killed.

Psychosis Defined

A 2D topdown roguelite inspired by games like minecraft and and risk of rain, the players goal is to descend into portals to take on infinite hoards of enemies to collect all sorts of goodies to take back to base where the player will be able to build houses, upgrade stations, recruit NPCs, and craft items all to help them to prepare to delve deeper into the portals and survive there longer.

Cooking Innana

A application built to learn more about 3D programming and mathematics, it started as a 100 day coding challenge during the pandemic with the goal of seeing what is involved in making a game. what I ended up with was a simple application that can load models, free camera movement, own lighting shaders, object selection, and the beginings of a physics system.

Assorted Projects

A collection of some simple python applications that range from applications I've made to help get a better understanding of a subject like ray-tracing, drawings fractals, and Machine Learning. Also included are some tools to help with other projects like the map maker I made for my 2D game Psychosis Defined.

Tayside Reusers

A python application made for a reuse charity to help them process data they were collecting more efficiently. The charity focused on recycling large household items and then recorded their impact by estimating the amount of CO2 they were saving from being outputted into the atmosphere. the application was made to save a lot of manual re-entry of data into multiple systems.
