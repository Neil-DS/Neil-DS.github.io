Psychosis Defined

itch.io

Psychosis Defined was started as a hobby project during the pandemic to get exposed to the full cycle of game development in order to expose my own weaknesses, so I could spend the time during lockdown working towards something productive. The driving goal behind Psychosis Defined was to complete a project using as few libraries as possible, so I could learn by having to write and implement my own versions of the systems needed.

During the project lots of small and big challenges presented themselves, design decisions for how to structure code boundaries, holes in my knowledge about newer programming features(c++ 17) learning vulkan while trying to create a full fledge game with it, while still learning about the graphics pipeline as a whole

more indepth understanding of compiling and building projects and dependencies, the beginnings of a physics system, with game update and frame updates happening at different intervals,
